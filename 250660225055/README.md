# 🌐 Implementasi Pembelajaran GitHub dan Pemrograman Web Berbasis Front End

Repository ini berisi hasil implementasi pembelajaran GitHub dan pemrograman web berbasis front end menggunakan HTML, CSS, JavaScript, dan React JS untuk membangun website modern, responsive, dan interaktif.

---

# 🎯 Tujuan Pembelajaran

Pembelajaran ini bertujuan untuk memahami:
- Penggunaan Git dan GitHub sebagai version control
- Dasar pengembangan website front end
- Struktur website menggunakan HTML
- Styling modern menggunakan CSS
- Interaksi website menggunakan JavaScript
- Pengembangan frontend modern menggunakan React JS
- Pembuatan website responsive dan user friendly

---

# 🛠️ Teknologi yang Dipelajari

## 1. Git & GitHub

Digunakan untuk version control dan manajemen repository project.

Materi:
- Git init
- Git clone
- Git add
- Git commit
- Git push
- Git pull
- Branch
- Pull Request
- Repository management

Contoh:

```bash
git init
git add .
git commit -m "first commit"
git push origin main
```

Manfaat:
- Menyimpan histori project
- Backup source code online
- Mempermudah kolaborasi
- Mengelola perubahan project

---

## 2. HTML5

Digunakan untuk membuat struktur dasar website.

Materi:
- Struktur dasar HTML
- Heading dan paragraph
- Link dan navigation
- Form input
- Image dan multimedia
- Section dan semantic HTML

Contoh:

```html
<h1>Portfolio Saya</h1>
<p>Frontend Developer</p>
```

---

## 3. CSS3

Digunakan untuk mempercantik tampilan website.

Materi:
- Typography
- Color dan background
- Flexbox
- CSS Grid
- Responsive design
- Media query
- Animation dan transition
- Hover effect
- Dark mode styling

Contoh:

```css
.hero {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

---

## 4. JavaScript

Digunakan untuk membuat website lebih interaktif.

Materi:
- DOM manipulation
- Event listener
- Form validation
- Dark / Light mode
- Local Storage
- Popup notification
- Portfolio filtering
- Loading animation

Contoh:

```javascript
button.addEventListener("click", function () {
  document.body.classList.toggle("dark-mode");
});
```

---

## 5. React JS

Digunakan untuk membangun frontend modern berbasis component.

Materi:
- JSX
- Component
- Props
- State
- Event handling
- React Hooks dasar
- Struktur project React
- Vite setup

Contoh:

```jsx
function App() {
  return (
    <h1>Hello React</h1>
  )
}

export default App
```

Keuntungan:
- Reusable component
- Faster rendering
- Struktur project lebih rapi
- Mudah maintenance
- Modern frontend framework
