# ⚙️ Cara Menjalankan Project

## 1. Clone Repository

```bash
git clone https://github.com/username/portfolio-project.git
```

## 2. Masuk ke Folder Project

```bash
cd portfolio-project
```

## 3. Jalankan dengan Live Server

Buka menggunakan:
- VS Code
- Extension Live Server

Klik kanan:

```text
index.html
```

Lalu pilih:

```text
Open with Live Server
```
