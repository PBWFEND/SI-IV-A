Saya membuat web portofolio ini menggunakan Tailwind CSS yang dijalankan melalui Node.js. Dalam proses pengembangannya, saya menggunakan struktur project yang sudah disediakan, sehingga konfigurasi awal seperti setup folder dan build system sudah tersedia dan dapat langsung dikembangkan sesuai kebutuhan.

Untuk pengelolaan file aset, saya menggunakan folder dist sebagai hasil build akhir dari project. Di dalam folder dist tersebut terdapat folder image yang berfungsi untuk menyimpan seluruh gambar atau media yang digunakan pada website portofolio ini.

Dengan struktur ini, proses pengembangan menjadi lebih terorganisir karena file sumber dan hasil dipisahkan dengan jelas.
